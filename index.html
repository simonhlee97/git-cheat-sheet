Super-Condensed Version Cheat-Sheet of Fabio Pacific's great FreeCodeCamp post

GIT COMMAND DESCRIPTION
git config -l check your git configuration
git config --global user.name "Fabio" configure your git username
git config --global user.email "john@example.com" setup your Git user email
git config --global credential.helper cache cache your login credentials
git init initialize a repo
git add filename_here add an individual file to staging area
git add . add all files to staging area
git add fil* add all files starting with 'fil' to staging area
git status show the status of the current repository
git commit commit changes, and open editor to write a full commit message
git commit -m "your commit message here" commit changes with a message
git commit -a -m "your commit message here" commit changes (and skip the staging area)
git log shows commit history for the current repo
git log -p shows commit's history including all files and their changes
git show commit-id see a specific commit
git log --stat shows stats about the changes in each commit, including line(s) changed and file names
git diff see changes made before committing them using "diff"
git diff --staged diff with the --staged flag to see any staged changes
git add -p see changes using "git add -p"
git rm filename remove tracked files from the current working tree
git mv oldfile newfile This command stages the changes, then it expects a commit message
git checkout filename revert unstaged changes
git reset HEAD filename
git reset HEAD -p 		revert staged changes
git commit --amend modify and add changes to the most recent commit
git revert HEAD revert the latest commit by using the head alias
git revert comit_id_here rollback an old commit
git branch branch_name create a new branch
git checkout branch_name switch to a newly created branch
git branch view list of branches
git checkout -b branch_name create and switch to a new branch
git branch -d branch_name delete a branch
git merge branch_name merge two branches
git log --graph --oneline show the commit log as a graph
git log --graph --oneline --all show the commit log as a graph of all branches
git merge --abort abort a conflicting merge (throw a merge away and start over)
git add remote https://repo_here add a remote repository
git remote -v see all remote repositories for your local repository
git remote show origin get more info about a remote repo
git push push changes to a remote repo
git pull retrieve the latest changes made to the remote repository
git branch -r shows the name of all remote branches that Git is tracking
git fetch fetch remote repo changes
git log origin/main check the current commits log of a remote repo
git merge origin/main merge a remote repo with your local repo
git remote update get the contents of remote branch w/o automatically merging
git push -u origin branch_name push a new branch to a remote repo
git push --delete origin branch_name remove a remote branch
git rebase branch_name transfer completed work from one branch to another
git rebase -i master run git rebase interactively using the -i flag
git push -f force a push request
git merge --squash feature to combine all the commits when merging a branch, use --squash
